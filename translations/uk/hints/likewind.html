Вам необхідно написати цикл, який би послідовно проходив по словнику з матросами та додавав би їх на один з двох кораблів залежно від віку. З'ясувати, як працювати зі словниками, вам може допомогти офіційна документація Python.
Чи ви могли би навести приклад коду?
Цикл та умова розподілу матросів будуть виглядати таким чином: 
for i in sailors.keys():
    if sailors[i] > 40 or sailors[i] < 20:
        #send sailor to the first ship
    else:
        #send sailor to the second ship
Яким чином мені записувати кожного конкретного матроса на потрібний корабель?
Для цього можна створити перед циклом 2 списки - first_ship & second_ship та використати метод append, щоб додавати матроса в потрібний список.
Мені потрібен приклад коду.
У поєднанні з попередньою частиною рішення це буде виглядати наступним чином: 
first_ship = []
second_ship = []

for i in sailors.keys():
    if sailors[i] > 40 or sailors[i] < 20:
        first_ship.append(i)
    else:
        second_ship.append(i)
Але імена матросів у списку ідуть у хаотичному порядку. Як мені їх відсортувати?
Для сортування можна використати функцію sorted() або метод sort(), який описаний у списку доступних методів для структури даних - list.
