Я не знаю з чого почати вирішення цієї місії.
Вам необхідно написати цикл, який би послідовно проходив по словнику з матросами та додавав би їх на один з двох кораблів залежно від віку.
Чи ви могли би навести приклад коду?
Цикл та умова розподілу матросів будуть виглядати таким чином: 
    for (let name in sailors) {
        if (sailors[name] < 20 || sailors[name] > 40) {
            //send sailor to the first ship
        } else {
            //send sailor to the second ship
        }
    }
Яким чином мені записувати кожного конкретного матроса на потрібний корабель?
Для цього можна створити перед циклом 2 списки - first_ship & second_ship та використати метод push(), щоб додавати матроса в потрібний список.
Мені потрібен приклад коду.
У поєднанні з попередньою частиною рішення це буде виглядати наступним чином: 
    let first = [], second = [];
    for (let name in sailors) {
        if (sailors[name] < 20 || sailors[name] > 40) {
            first.push(name);
        } else {
            second.push(name);
        }
    }
Але імена матросів у списку ідуть у хаотичному порядку. Як мені їх відсортувати?
Для сортування можна використати метод sort().
