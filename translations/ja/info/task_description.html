冒険へようこそ！ 今あなたは危険であると同時に大変エキサイティングな旅を始めました。 道中多くの謎を解き、罠を避け、追跡者を出し抜く必要があります。 これら全てはこの世界で最も価値あるアーティファクト、無限エネルギーの源「ハイパーキューブ」を見つけ出すための唯一の手段です。

ハイパーキューブは変わり者のエッシャー卿が数世紀前に建てた城の中心に配置されてます。 彼は孤独への偏愛ゆえに島に自身の城を建てました。 彼の (実に謎多き) 死後、島から大陸に帰還した従者よりこの世界の奇妙な物事や現象が語られ、多くの探検隊とトレジャーハンターのグループが島を訪れましたが帰還者はいませんでした…。

幸運を祈ります! 多分あなたは他の人たちより運がいいはずです。

装備を集め荷詰めをし出発の時が来ました。 現在あなたは大陸の港にいます。 あなたは島で直面する危険を考慮して船員を雇い 2つの船を準備しました。 あなたの目的はキューブを入手することであり、道中で発見するであろう他の宝物は船員への報酬にするつもりです。

最初の仕事は簡単です。あなたは船員を 2つのチームに分けなくてはなりません。(船は 2隻あるのですから)

あなたは次のルールで船員を 2つのチームに分ける必要があります: 40歳以上もしくは 20歳以下の船員からなる第1チーム、その他の船員の第2チーム。 これは熟練船員が若手の船員を助けるためです。 入力データは船員の姓をキーとしその年齢を値とする辞書です。 船員の振り分け後、各チームの姓のリストをアルファベット順でソートすること。

入力: 船員の姓と年齢の辞書

出力: 2つのチームのリストのリスト
